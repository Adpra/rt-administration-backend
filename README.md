## How to install

**Use PHP version 8.1 / 8.3**

- Set `.env` like `.env.example`
- Run `composer install`
- Run `php artisan migrate --seed`
- Run `php artisan storage:link`
- Run `php artisan serve`

**MySQL**

I'm using database version 8.2.0.
